Job Apply for Claude Web Chat

Generate tailored cover letters and resumes using Claude's free web chat. No install required.

Download Files

2 small files — SKILL-webchat.md + generate_word_docs_web.py

Setup (5 minutes)

  1. Download the 2 files using the button above
  2. Go to claude.ai and create a new Project
  3. Add SKILL-webchat.md as Project instructions (paste its contents into the custom instructions field)
  4. Add generate_word_docs_web.py to Project knowledge (upload the file)
  5. Start a conversation — you're ready to go

Your First Prompt

Here's my resume and the job description for [Role] at [Company]. Please assess my fit and generate tailored application documents.

Then attach your resume (.docx or .pdf) and paste the full job description.

What to Expect

  • You upload your resume and paste a job description
  • Claude scores your fit, shows strengths and gaps, and asks if you want to proceed
  • You say "Yes"
  • Claude generates .docx cover letter + resume for download

3–4 messages total per application.

Returning Users

After your first run, Claude offers a profile.yaml file. Save it. Next time, upload profile.yaml instead of your resume to skip parsing and save a message.
