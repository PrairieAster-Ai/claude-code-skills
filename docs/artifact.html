How to Use

  1. Open the HTML file in any modern browser (Chrome, Firefox, Safari, Edge)
  2. Paste a job description into the left textarea and your resume into the right textarea
  3. Click Analyze Fit to see your score, strengths, gaps, and recommendation
  4. Review your tailored cover letter and resume, pick a style, and download as .doc
Your inputs are auto-saved to browser localStorage. Close the tab, come back later, and your work is still there. Click Clear Saved Data to start fresh.

How the Assessment Works

The fit assessment uses heuristic NLP — no AI model needed. It extracts requirements from the job description, matches them against your resume, and calculates a weighted score.

Scoring

Component Weight How It’s Measured
Must-Have Requirements 70% Keyword overlap, synonym matching, Jaccard similarity per bullet
Nice-to-Have Requirements 30% Same matching engine, lower weight

Recommendation Tiers

Tier Overall Must-Haves
Strong Fit 80%+ 80%+
Good Fit 60–79% 70%+
Stretch Fit 50–59% 60%+
Poor Fit < 50% < 60%
Disagree with the assessment? Click any match icon to cycle between Strong, Partial, and Gap. Click skill chips to move them between matched and missing. Then click Update Scores & Documents to recalculate.

Frequently Asked Questions

Does this send my data anywhere?
No. Everything runs locally in your browser. No network requests are made. Your job descriptions and resume text never leave your machine.
What’s the difference between .doc and .docx?
The artifact exports .doc files using an MHTML format that Word can open. They work in Microsoft Word and LibreOffice but may look slightly different than native .docx. For pixel-perfect Word documents, use the CLI or Web Chat edition.
Can I use a fun preset for a real application?
The fun presets produce complete, functional documents with all the right sections. Whether to actually submit one is between you and your risk tolerance. They work great for internal transfers, creative industry roles, or just brightening someone’s day.
How accurate is the fit scoring?
The scoring uses keyword matching, synonym lookup, and Jaccard similarity — not an AI model. It’s a fast, deterministic approximation. If something looks wrong, use the interactive overrides to correct it and the scores will recalculate instantly.
My data disappeared — what happened?
Your inputs are saved to localStorage, which is per-browser and per-origin. Clearing browser data, using incognito mode, or opening the file from a different location will start fresh. Click Clear Saved Data intentionally to reset.
Can I use this as a Claude web artifact?
Yes. The HTML file was designed to work both as a standalone page and as a Claude web chat artifact. Ask Claude to create an artifact, then paste the HTML content.
