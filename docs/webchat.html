Quick Start

Free Tier Free

  1. Download SKILL-webchat.md and generate_word_docs_web.py from above
  2. Open claude.ai and start a new conversation
  3. Attach SKILL-webchat.md, your resume (PDF or text), and the job description
  4. Send the prompt below — Claude will generate the fit assessment and documents
  5. Copy the Python output into generate_word_docs_web.py and run it to create .docx files

Your First Prompt

Paste this into your Claude conversation after attaching your files:

Follow the instructions in SKILL-webchat.md. Analyze the attached resume against the job description. Produce the fit assessment, then generate a tailored cover letter and resume. Output the Python code block I can paste into generate_word_docs_web.py to create the .docx files.

How It Works

  1. Attach & Prompt

    Upload your resume and job description to Claude with the skill file. One prompt kicks off the analysis.
  2. AI Analysis

    Claude reads both documents, scores must-have and nice-to-have requirements, and identifies strengths and gaps.
  3. Document Generation

    A tailored cover letter and resume are generated with skills reordered and bullets prioritized for the role.
  4. Export to .docx

    Copy the Python output into the helper script and run it. Polished Word documents land in your working directory.

Returning Users

Save time with profile.yaml — If you use this skill regularly, create a profile.yaml with your name, contact info, and key skills. Attach it alongside your resume so Claude can pre-fill your details without re-reading the full document every time. This speeds up analysis and keeps your output consistent across applications.

Troubleshooting

  • Claude doesn't follow the skill instructions

    Make sure SKILL-webchat.md is attached as a file, not pasted inline. Claude reads attached files as structured instructions. If using a Project, add it to project knowledge instead.

  • The Python script fails or produces empty files

    Ensure you have the python-docx package installed (pip install python-docx). Copy the entire code block from Claude's output — partial copies may be missing closing statements or data definitions.
